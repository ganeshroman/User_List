# User List Demo


## Summary
- **Summary:**  Android App showing list of Users, Search/Filter by name and show details on Click of List Item
- **Created Date:** 28 June 2023
- **Last modified:** 28 June 2023 



### Technology Used
- Kotlin
- MVVM
- Retrofit
- Data Binding
- Dialog Fragment
- RecyclerView
- Card View

### Backend Details

- **Webservice** : https://gorest.co.in/public/v2/users
